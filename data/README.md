# Data Files

## English

### Canonical (Current)

| File | Description |
|------|-------------|
| **`mapping_v7.json`** | **THE SOLUTION** — 98 two-digit codes mapped to 22 German letters (94.6% coverage) |
| `books.json` | All 70 Hellgate Library books as digit strings (11,263 digits total) |
| `bookcase_mapping.json` | Which book sits on which bookcase (40 bookcases, 71 slots) |
| `decoded_narrative.json` | Full decoded text with per-character coverage tracking |
| `decoded_text.txt` | Plain text decoded output (human-readable German) |
| `chain_info.json` | 12 chain reconstructions + 12 isolated books from overlap analysis |
| `book_positions.json` | In-game physical position metadata |
| `master_text.txt` | Reconstructed continuous narrative from all chains |

### Archive (Historical)

The `archive/` folder contains superseded mapping versions (v1 through v6). These document the mapping evolution across 30 sessions but are not needed for decoding.

| File | Coverage | Session |
|------|----------|---------|
| `final_mapping.json` | ~40% | Sessions 1-4 |
| `final_mapping_v2.json` | ~50% | Sessions 5-6 |
| `final_mapping_v3.json` | ~55% | Sessions 7-8 |
| `final_mapping_v4.json` | ~60% | Sessions 9-12 |
| `mapping_v6_hybrid.json` | ~81% | Sessions 13-24 |
| `best_mapping.json` | ~81% | Best intermediate result |
