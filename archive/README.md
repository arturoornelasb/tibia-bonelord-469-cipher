# Archive

This directory preserves historical research artifacts that were part of the investigation
but are no longer needed to reproduce the final results.

## Contents

| Folder | Description |
|--------|-------------|
| `scripts/experimental/` | Early-stage hypothesis scripts (base-5, bigram, DNA alignment, etc.) |
| `scripts/sessions/` | Per-session crack scripts from Sessions 9–12 attack rounds |
| `scripts/core_misc/` | One-off investigation and intermediate pipeline scripts |
| `scripts/analysis/` | Per-session analysis scripts from Sessions 18–31 |
| `agente3/` | Spanish-language investigation phase (separate research track) |

## Note

Nothing in this directory is needed to decode the cipher or reproduce the main results.
The canonical reproduction path is documented in the root `Makefile` and `README.md`.
